## 🚀 快速部署指南

### 方案A: 浏览器脚本部署（推荐，快速实施）

1. **安装用户脚本管理器**
   - Chrome: 安装 Tampermonkey 扩展
   - Firefox: 安装 Greasemonkey 扩展

2. **部署脚本**
   ```bash
   # 1. 打开脚本管理器
   # 2. 点击"添加新脚本"
   # 3. 复制 customer_service_access_blocker.user.js 内容
   # 4. 保存并启用脚本
   ```

3. **配置用户角色识别**
   ```javascript
   // 修改脚本中的 getUserRole() 函数
   function getUserRole() {
       // 方法1: 从页面元素获取
       const userInfo = document.querySelector('.user-role');
       if (userInfo) return userInfo.textContent;
       
       // 方法2: 从URL参数获取（测试用）
       const params = new URLSearchParams(window.location.search);
       return params.get('user-role') || 'unknown';
   }
   ```

4. **测试验证**
   ```bash
   # 客服账号访问测试URL:
   https://sellercentral.amazon.com/sw/AccountInfo/?user-role=客服专员
   
   # 应该看到阻断页面
   ```
