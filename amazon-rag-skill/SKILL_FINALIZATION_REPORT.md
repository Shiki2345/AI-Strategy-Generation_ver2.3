### **决策规则**
| 用户表达 | 系统判断 |
|----------|----------|
| "需要审批" | → 审批制访问控制 |
| "全天生效" | → 设置 7x24 时间范围 |
| "由 BOSS 定义" | → duration_controlled_by: approver |
| "可选填写" | → reason_required: false |

1. **审批超时 30 分钟** - 平衡及时性和用户体验
2. **申请理由可选** - 降低门槛但保留审批依据
3. **多渠道通知** - 确保审批人及时响应
4. **访问期间倒计时** - 防止用户忘记权限到期
5. **完整审计日志** - 满足合规要求
6. **自动过期撤销** - 避免权限滥用
