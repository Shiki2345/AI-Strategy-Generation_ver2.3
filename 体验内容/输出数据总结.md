# 输出数据内容要求

## 一、共有部分（所有策略必填）

### 1. 策略名称 ⭐必填
字段名：`strategy_name`  
说明：请输入访问策略名称  
限制：0-24字符

### 2. 生效成员 ⭐必填
字段名：`effective_members`  
说明：指定具体范围（可多选）  
示例："除BOSS外所有成员" 或 指定成员列表

### 3. 生效平台账号 ⭐必填
字段名：`effective_accounts`  
说明：选择账号（可多选）  
示例："所有账号" 或 指定账号列表

### 4. 生效时段 ⭐必填
字段名：`effective_period`  
选项：
- 永久生效
- 指定时段生效（需配置以下内容）：
  - 生效日期：开始时间 → 结束时间
  - 生效周期：具体到星期几（1-7）
  - 生效时间：开始时间 → 结束时间

### 5. 访问策略描述 ⚪选填
字段名：`strategy_description`  
限制：0-400字符

---

## 二、策略类型（二选一）

### 1. 指定网页生效策略

以下规则将对指定网页的访问行为生效：

**网页访问模式**：黑名单模式  
字段名：`access_mode`  
值：`"blacklist"`

说明：
- 添加禁止访问的网页后，这些网页将受到限制
- 其他网页可以正常访问
- 提示：新手建议使用此模式

**访问审批限制**：
- 仅boss账号可审批：字段 `only_boss_approve`（true/false）
- 不允许申请访问：字段 `disallow_request`（true/false）

---

### 2. 所有网页通用策略

以下规则将对账号环境内所有网页下的操作行为生效：

**1. 复制**  
字段名：`allow_copy`  
说明：若限制复制，将无法复制账号环境内的信息  
操作选项：
- "allowed" - 允许
- "allowed_and_logged" - 允许且记录（默认）
- "restricted_and_logged" - 限制且记录

**2. 文件上传**  
字段名：`allow_upload`  
说明：若限制文件上传，将无法在账号环境下执行文件上传操作  
操作选项：
- "allowed" - 允许
- "allowed_and_logged" - 允许且记录（默认）
- "restricted_and_logged" - 限制且记录



**3. 文件下载**  
字段名：`allow_download`  
说明：若限制文件下载，将无法在账号环境下执行文件下载操作  
操作选项：
- "allowed" - 允许
- "allowed_and_logged" - 允许且记录（默认）
- "restricted_and_logged" - 限制且记录



**4. 打印**  
字段名：`allow_print`  
说明：若限制打印，将无法在账号环境下执行打印操作  
操作选项：
- "allowed" - 允许
- "allowed_and_logged" - 允许且记录（默认）
- "restricted_and_logged" - 限制且记录



**5. 开发者模式**  
字段名：`allow_devtools`  
说明：若限制使用开发者模式，将无法在账号环境下使用「F12」等快捷键开启开发者模式；如，无法通过开发者模式查看店铺账号密码等信息（当前仅支持控制Chrome浏览器）  
操作选项：
- "allowed" - 允许
- "allowed_and_logged" - 允许且记录（默认）
- "restricted_and_logged" - 限制且记录



**6. 查看密码框**  
字段名：`allow_view_password`  
说明：若限制查看密码框，生效的成员将无法在账号环境下查看密码框的明文；如，无法查看「密码」、「密钥」等机密信息  
操作选项：
- "allowed" - 允许
- "restricted" - 限制（默认）

## 三、JSON数据结构示例

### 示例1：指定网页策略（黑名单模式）
```json
{
  "strategy_name": "工作时间网页限制",
  "effective_members": "除BOSS外所有成员",
  "effective_accounts": "所有账号",
  "effective_period": "指定时段生效",
  "effective_date": {
    "start_date": "2024-01-01 09:00",
    "end_date": "2024-12-31 18:00"
  },
  "effective_weekly": [1, 2, 3, 4, 5],
  "effective_time": {
    "start_time": "09:00",
    "end_time": "18:00"
  },
  "strategy_description": "工作时间禁止访问社交媒体",
  "strategy_type": "specific_website",
  "access_mode": "blacklist",
  "blocked_websites": [
    "https://www.youtube.com",
    "https://www.facebook.com"
  ],
  "only_boss_approve": false,
  "disallow_request": false
}
```
